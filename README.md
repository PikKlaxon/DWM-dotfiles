## quick-start

to get started run these commands:

```
git clone https://github.com/PikKlaxon/dwm_fork
cd dwm_fork
cd dmenu
sudo make clean install
cd..
cd dwm
sudo make clean install
cd..
cd slock
sudo make clean install
cd ..
cd slstatus
sudo make clean install
cd .. 
cd st
sudo make clean install
```

then add this to your `~/.xinitrc`

```
slstatus &
exec dwm
```
